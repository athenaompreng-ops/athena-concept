# 📋 Administrasi — Dokumen Resmi ATHENA

Folder ini berisi seluruh dokumen administrasi resmi yang digunakan dalam proses pengajuan dan evaluasi proyek **ATHENA** pada kompetisi **FIKSI 2026**.

---

## Daftar Dokumen

| No | Dokumen | Format | Deskripsi |
|----|---------|--------|-----------|
| 1 | [Deskripsi Produk](./ATHENA_Deskripsi_Produk.pdf) | PDF | Dokumen utama deskripsi produk ATHENA untuk FIKSI 2026. Mencakup ringkasan eksekutif, problem statement, solusi teknis 4A Shield, model bisnis, dan roadmap. |
| 2 | [Analisis SWOT](./ATHENA_Analisis_SWOT.pdf) | PDF | Analisis SWOT komprehensif platform ATHENA — strengths, weaknesses, opportunities, dan threats beserta strategi mitigasi. |
| 3 | [Channel Strategy](./ATHENA_Channel_Strategy.pdf) | PDF | Strategi distribusi dan channel pemasaran ATHENA untuk menjangkau segmen target: kreator konten, pengrajin UMKM, dan pengguna individu. |
| 4 | [HPP Produk](./ATHENA_HPP_Produk.xlsx) | XLSX | Harga Pokok Produksi per unit (per 1.000 foto diproses), mencakup biaya GPU inference, storage, bandwidth, dan overhead backend. |
